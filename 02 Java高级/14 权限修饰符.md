# 权限修饰符

权限修饰符总共有四种，权限从大到小的顺序为

- `public`
- `protected`
- `(default)`，就是什么都不写
- `private`

下表列出了不同修饰符下的访问规则

|             | 同一个类 | 同一个包 | 不同包的子类 | 不同包非子类 |
| ----------- | -------- | -------- | ------------ | ------------ |
| `public`    | 能       | 能       | 能           | 能           |
| `protected` | 能       | 能       | 能           | **不能**     |
| `(default)` | 能       | 能       | **不能**     | **不能**     |
| `private`   | 能       | **不能** | **不能**     | **不能**     |
