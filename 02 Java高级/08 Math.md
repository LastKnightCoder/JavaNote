# Math

`Math`类位于`java.util`包中，该类包含了很多与数学计算相关的静态方法。这里介绍几个与整数操作有关的方法

- `abs()`
  - 取绝对值
  - `abs(-2) = 2`
- `ceil()`
  - 向上取整
  - `ceil(2.1) = 3`，`ceil(-2.1) = -2`
- `floor()`
  - 向下取整
  - ·`floor(2.1) = 2`，`floor(-2.1) = -3`
- `round()`
  - 四舍五入
  - `round(2.1) = 2`，`round(-2.1) = -2`

当然`Math`类还包含很多的方法，具体的可以查阅资料。
