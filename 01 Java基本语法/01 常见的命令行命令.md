# 常见的命令行命令

快捷键`Win+R`，输入`cmd`进入命令行。

下面介绍一些常见的命令行命令：

- 切换盘符(下面以切换到D盘为例)

```powershell
D:
d:
```

- 显示当前路径下的目录结构

```powershell
dir
```

- 清屏

```powershell
cls
```

- 退出命令行

```powershell
exit
```

- 切换路径

```powershell
cd 路径(可以是绝对路径，也可以是相对路径)
```

**注意几个特殊的路径：**

- `\`：根路径
- `.`：当前路径
- `..`：上一级路径，根路径的上一级路径还是根路径
